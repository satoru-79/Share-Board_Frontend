# Share-Board 
## URL
https://shareboard-26307.firebaseapp.com
### 概要
　バスケットボールの作戦版をデジタル化し、作成したボードを共有・保存ができるサイトです。
### 使用技術
* React 18.2.0 <br>
 * TypeScript 4.9.5 <br>
 * Tailwind CSS 3.4.3 <br>
 * node.js 20.11.0<br>
 * FireBase 10.11.0<br>
   ・ Authentication <br>
   ・ Firestore Database <br>
   ・ Hosting <br>

### 機能
   * Googleアカウントによるユーザー登録、ログイン機能
   * デジタル化した作戦版を自由に操作できる機能 
   * 作成したボードを投稿する機能
   * 投稿されたボードへのいいね機能、保存機能
